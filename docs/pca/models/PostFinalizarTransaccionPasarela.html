PostFinalizarTransaccionPasarela

  • id : String
    id de la transaccion transaccion
  • estado : String
    estado de la transaccion al finalizar (No puede ser EN_PROCESO)
    Enum:
    APROBADO
    EN_PROCESO
    CANCELADO
    RECHAZADO
  • requiereFactura : Boolean
    cliente requiere factura electronica (si/no)
  • jsonResultado : String
    json del resultado de la transaccion
